# Talkto Connect

A real-time chat and calling app built with Flask, Socket.IO, and WebRTC.

## Features
- Live messaging without refresh
- Typing indicators and presence
- Secure login/register
- Peer-to-peer video calling

## Setup
1. Clone the repo
2. Create `.env` from `.env.example`
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python app.py`

## Deployment
Use platforms like Render, Railway, or Heroku. Make sure to enable WebSocket support.
